BACKEND INSTALLATION
====================

1. Create virtual environment in back/:
   python -m venv .venv

2. Activate:
   Windows: .venv\Scripts\activate
   Linux/Mac: source .venv/bin/activate

3. Install dependencies:
   pip install -r requirements.txt

4. Train model:
   python -m core.train

5. Run API:
   python api.py
   # API: http://localhost:8000
