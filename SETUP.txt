BRAIN TUMOR CLASSIFICATION WITH XAI
====================================

STRUCTURE:
----------
back/       - Backend (FastAPI + PyTorch)
  core/     - Core ML modules (model, dataset, xai, train)
  api.py    - FastAPI server
  utils.py  - Helper functions
front/      - Frontend (React + Vite)
  src/      - React components & logic
models/     - Trained models
data/       - Dataset (raw & processed)
notebooks/  - Jupyter notebooks

INSTALLATION:
-------------
1. Backend:
   cd back
   pip install -r requirements.txt

2. Frontend:
   cd front
   npm install

TRAINING:
---------
cd back
python -m core.train

RUNNING:
--------
1. Backend (Terminal 1):
   cd back
   python api.py
   # Runs on http://localhost:8000

2. Frontend (Terminal 2):
   cd front
   npm run dev
   # Runs on http://localhost:3000

FEATURES:
---------
- Transfer Learning with ResNet50
- XAI methods: Grad-CAM, SHAP, LIME
- 4 classes: no_tumor, glioma, meningioma, pituitary
- Real-time predictions with explanations
